# 문서한장

견적서·발주서·거래명세표를 간편하게 만들고, 필요할 때 사진과 직인을 첨부하는 웹앱입니다.

## GitHub Pages 배포

1. GitHub에서 새 Public 저장소를 만듭니다.
2. 이 폴더 안의 파일들을 저장소 최상단에 업로드합니다.
3. 저장소 `Settings` → `Pages`
4. `Source`를 `Deploy from a branch`로 선택
5. `main` 브랜치와 `/(root)`를 선택하고 저장
6. 표시되는 GitHub Pages 주소를 휴대폰에서 엽니다.

업체 정보와 직인은 사용 중인 브라우저 기기에 저장됩니다.
